# Architecture Decision Records (ADR)

Karpeta honek DIY Garajea proiektuan hartutako erabaki arkitektoniko nagusiak jasotzen ditu.

ADR bakoitzak honakoak azaltzen ditu:

- erabaki bat hartu den testuingurua
- aztertutako aukerak
- hartutako erabakia
- ondorio nagusiak

ADR-ak dokumentazio osagarriak dira, eta ez dute negozio-logika edo inplementazio xehetasunik jasotzen.

## ADR zerrenda

- ADR-001: Service Layer erabiltzea
- ADR-002: Zerbitzuak kasu-erabileren arabera diseinatzea
